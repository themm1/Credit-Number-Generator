What is CCGV

CCGV (Credit Card number Generator and Validator) is a basic web app built with Flask for generating and validating credit card numbers.

Why is this usefull

There are two main reasons for using such an app.

  • Some websites want your credit card number even if you aren't paying. We believe there's no need to share such information and this is the way to protect yourelf in such a situations.
  • The other reason we made this are programmers testing ecommerce websites, applications or other software. For this kind of use we are making also advanced generator, which is currently unavailable.

How do we generate valid credit card nubmer

For generating valid credit card number we need to check for some requirements

  • first digits of the number
  • length of the number
  • valid Luhn algorithm

Luhn algorithm (MOD 10 alogrithm)

The Luhn algorithm is a checksum formula which is used for validating credit card numbers.

Say we have number 38520000023237

  • double every other digit starting from the right
  • 6, 6, 0, 0, 0, 10, 6
  • take the sum of the numbers, if numbers contains two digits, sum the digit products
  • 6 + 6 + 0 + 0 + 0 + 1 + 0 + 6 = 19
  • now sum the rest of the digits
  • 7 + 2 + 2 + 0 + 0 + 2 + 8 = 21
  • add our two sums
  • 19 + 21 = 40
  • if the final sum is devisible by 10, Luhn algorithm is valid
  • 40 / 10 = 4, valid
